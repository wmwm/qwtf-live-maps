# Licensing & attribution

The maps in this repo are third-party community content, built by many
different QuakeWorld Team Fortress mappers over roughly two decades. This
repo does not claim authorship or ownership of any map asset.

**Stance: attribution-only, scene convention.** This follows the long-standing
norm in the QuakeWorld/QWTF community — maps are freely shared and mirrored
between servers, packs, and archives (see `FortressOne/map-repo` for the
long-running precedent). Where a map's original author is known, credit
them in that map's `map.yml` (`author:` field). Most entries currently read
`author: unknown` — this is a real gap, not a placeholder; if you know who
built a map in this repo, a PR correcting it is welcome.

This is **not** a formal per-map provenance/rights audit. If you are the
original author of a map here and want it credited differently, removed, or
handled another way, please open an issue.

Contributed code in this repo (the harvesting/loc-generation/metadata
scripts under `scripts/`) is not subject to the above — that's original
tooling, MIT-licensed unless noted otherwise.
